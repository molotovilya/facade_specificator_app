# 📘 Спецификатор фасадов

Десктопное приложение на Windows для создания спецификаций и расчёта стоимости производства фасадов.

## 📋 Описание

Программа предназначена для автоматизации создания спецификаций на производство мебельных фасадов. Позволяет вести учёт проектов, рассчитывать материалы, стоимость работ и формировать готовые документы для печати.

## ✨ Основной функционал

- **Управление проектами**: сохранение, загрузка, автосохранение
- **Работа с таблицами**: 
  - Ввод параметров деталей (длина, ширина, толщина, количество)
  - Поддержка различных типов фасадов (фрезерованные, мыло, шпон, комбинированные)
  - Умная сортировка по краске, типу и размеру
- **Автоматические расчёты**:
  - Площадь деталей и покраски
  - Расход краски
  - Дополнительные работы (ручки, склейка, замотка, оклейка)
  - Стоимость работ по настраиваемым ставкам
- **Настройка структуры**:
  - Добавление своих типов фасадов и работ
  - Гибкая система колонок
  - Сохранение исполнителей и ставок
- **Экспорт и печать**: 
  - Формирование PDF документов
  - Прямая печать спецификаций

## 🛠 Технологии

## 🚀 Установка и запуск

### Для пользователей

1. Скачайте релиз или клонируйте репозиторий
2. Перейдите в папку `app`
3. Запустите `app.exe`
4. При первом запуске автоматически создадутся необходимые папки, если их не было

### Для разработчиков

1. Клонируйте репозиторий
2. Исходный код проекта находится в папке `prj/`
3. Для сборки используйте скрипты из папки `build/`

## 📖 Использование

### Быстрый старт

1. **Создание проекта**
   - Номер проекта присваивается автоматически
   - Укажите заказчика и дату
   
2. **Заполнение таблицы**
   - Введите параметры деталей
   - Выберите тип фасада из списка
   - Укажите краску
   - Используйте навигацию: Enter, Tab, стрелки

3. **Сортировка**
   - Нажмите кнопку **SORT** после заполнения
   - Детали автоматически группируются по краске, типу и толщине

4. **Расчёт и сохранение**
   - Кнопка **Рассчитать** покажет итоговые данные
   - **Сохранить проект** - сохранение в рабочем формате
   - **Сохранить PDF** или **Печать** - готовый документ

### Клавиатурные сокращения

- `Enter` - сохранить и перейти вниз
- `Tab` - сохранить и перейти вправо
- `↑↓` - навигация между ячейками
- `Ctrl+C` / `Ctrl+V` - копировать/вставить
- `Ctrl+Z` - отмена действия
- `Delete` - удалить значение

## 🔧 Настройка

### Типы фасадов
Добавляйте свои типы в блоке "Структура":
- Обычные: `фр`, `мыло`, `шпон`, `перекрас мыло`
- Комбинированные: `фр+мыло`, `мыло+шпон` - делятся автоматически

### Ставки и расчёты
В блоке "Вывод" настраиваются:
- Ставки за работы для каждого типа фасада
- Расход краски
- Стоимость замотки и склейки

## 📄 Формат сохранения

Проекты сохраняются в формате: `номер_заказчик_дата`

Пример: `001_ИвановИ_2024-10-11`

## 🔄 Перенос программы

Для переноса на другой компьютер:
1. Скопируйте всю папку `app` со всем содержимым
2. Все настройки и проекты сохранятся

Для сброса нумерации проектов: очистите папку `save/projects`

## 🛠 Разработка

### Требования
- Python 3.x
- Дополнительные библиотеки (указаны в requirements.txt или в коде)
- не поддерживается ниже Wibdows 8

### Структура разработки
- `prj/` - исходный код проекта
- `build/` - скрипты для сборки приложения
- `save/` - временные файлы при разработке

### Сборка
```bash
cd build
python build.py
```

Готовое приложение появится в папке `app/`
